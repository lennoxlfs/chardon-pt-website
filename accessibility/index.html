Accessibility Statement

We want everyone to be able to use our website and visit our clinics. Here's what we're doing, and how to reach us if something needs work.

Chardon Performance Therapy is committed to providing an accessible experience for all users. This statement explains the standards we follow, what we've implemented, and how to contact us if you encounter a barrier.

Our commitment

We aim to align this website with Web Content Accessibility Guidelines (WCAG) 2.1 Level AA, the international standard set by the W3C. This covers perceivability, operability, understandability, and robustness — the four WCAG principles.

Measures we've taken

  • Keyboard navigation: every interactive element is reachable and operable without a mouse. A visible focus indicator appears when tabbing.
  • Skip-to-content link: available at the top of every page for keyboard and screen-reader users.
  • Semantic HTML: proper heading hierarchy, landmarks (header, nav, main, footer), and ARIA labels on interactive elements.
  • Color contrast: body text and UI controls meet or exceed WCAG AA contrast ratios against their backgrounds.
  • Descriptive alt text: informative images have text alternatives; decorative images are marked so screen readers can skip them.
  • Form accessibility: form fields have associated labels, required-field indicators are visible, and validation errors use live regions.
  • Motion sensitivity: we respect the operating-system prefers-reduced-motion setting and disable non-essential animation when requested.
  • Responsive design: content reflows to 320px viewports without horizontal scrolling or loss of functionality.
  • Text resizing: text can be enlarged up to 200% without loss of content or function.
  • Video and audio: where we publish multimedia, captions and transcripts are provided.

In-clinic accessibility

  • Both CPT locations are on the ground floor and ADA-accessible.
  • Free on-site parking is available directly in front of each entrance.
  • If you have specific accessibility needs for your visit, let us know when you book — we'll make sure everything is ready.

Known limitations

Despite our ongoing efforts, some areas of the site may not yet be fully accessible. Specifically: third-party embedded maps (Google Maps) rely on external rendering that we don't fully control. If you need an alternative to embedded content, please contact us — we'll provide the information directly.

Ongoing monitoring

Accessibility is an ongoing process, not a one-time project. We regularly review new content, test with screen readers and keyboard navigation, and incorporate user feedback.

Report a barrier

If you encounter an accessibility barrier on this website — anything that prevents you from using, reading, or interacting with content — please tell us. We take these reports seriously and fix issues quickly.

  • Email: team@chardonpt.com
  • Phone: (440) 299-7171
  • Mail: Chardon Performance Therapy, 520 Fifth Avenue, Suite #3, Chardon, OH 44024

When reporting, please include the page URL, a description of the issue, and any assistive technology you were using. We'll respond within three business days and work with you on a solution.

Compatibility

This website is designed to work with current versions of:

  • Chrome, Firefox, Safari, and Edge (desktop and mobile)
  • Screen readers including VoiceOver (iOS/macOS), TalkBack (Android), NVDA, and JAWS
  • Keyboard-only navigation
  • Browser zoom up to 200%

Standards reference

Our target is WCAG 2.1 Level AA. Conformance is assessed against the current version at w3.org/WAI/WCAG21/quickref.
